# Practice Deck — 音楽分析プレイヤー

音源の再生速度調整(音程保持)・ピッチ表示・A-Bループ練習ができる、
ブラウザだけで動く音楽分析プレイヤーです。

## フォルダの内容

- `index.html` — アプリ本体
- `manifest.json` — PWA(インストール可能なWebアプリ)用の設定
- `sw.js` — オフラインでも動かすためのService Worker
- `icon-192.png` / `icon-512.png` — アイコン

## 使い方

### そのまま使う(一番簡単)

`index.html` をダブルクリックしてブラウザで開くだけで動きます。
この場合、ホーム画面への追加やオフライン対応は機能しません(音楽分析の
機能自体は問題なく使えます)。

### スマホのホーム画面に追加して使う(おすすめ)

PWAとしてインストールするには、HTTPSまたはlocalhost経由で配信する必要が
あります。簡単な方法は以下のとおりです。

1. このフォルダをスマホやPCに置く
2. フォルダ内で以下を実行してローカルサーバーを立てる

   ```
   python3 -m http.server 8000
   ```

3. 同じネットワーク内のスマホのブラウザで `http://(PCのIPアドレス):8000/` を開く
4. ブラウザのメニューから「ホーム画面に追加」を選ぶ

これでアイコンからアプリのように起動できるようになり、2回目以降は
オフラインでも動作します。

### 注意

- 音声データは外部に送信されず、すべてその場(ブラウザ内)で処理されます。
- ピッチ検出は単音のメロディに最適化されています。和音や伴奏が
  混ざっていると音名表示が安定しないことがあります。
